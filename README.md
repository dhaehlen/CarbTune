# CarbTune

This is a project I am using to learn. However, hopefully the final product is usable as a tool to help me tune
my CB400F carburetors. Ideally the code and hardware will be able to tune carbs on other vehicles as well.

The development is being done largerly with Claude. The goal is to create a strong FSD and build the code from their.
For more information about the inspiration of the code development process see this video from Andreas Spiess: 
[Why AI Agents Replaced the Arduino IDE in My ESP32 Projects](https://www.youtube.com/watch?v=5DG0-_lseR4&t=1218s).

# Goals

- [ ] Learn about development with AI tools
- [ ] Learn about embedded development
- [ ] Learn about circuit and microcontroller design
- [ ] Learn about sensor integrations
- [ ] Build a useble tool for tuning caburetors
